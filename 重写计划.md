# AI Companion 项目重写计划

## 项目背景

当前项目是一点一点累加的，代码混乱、前后端混在一起、控制台是独立的 Tkinter 程序。用户希望：
1. **去掉 Tkinter 控制台**，所有管理功能集成到网页
2. **单服务架构**，一个 Python 进程处理所有功能
3. **修复通话模式**（VAD、声纹识别、打断机制）
4. **模块化代码**，前后端分离

## 新架构设计

```
ai-companion/
├── app.py                 # 主入口，启动 Web 服务器
├── config.py              # 配置管理（从 .env 加载）
├── db.py                  # 数据库操作（SQLite）
├── llm.py                 # LLM 调用（多后端统一接口）
├── tts.py                 # TTS 语音合成（Edge TTS）
├── asr.py                 # ASR 语音识别（SenseVoice）
├── call.py                # 通话模式（WebSocket + VAD + 声纹）
├── routes/
│   ├── __init__.py
│   ├── chat.py            # 聊天相关 API
│   ├── memory.py          # 记忆管理 API
│   ├── person.py          # 人物管理 API
│   ├── relation.py        # 关系管理 API
│   ├── admin.py           # 后台管理 API
│   └── call_ws.py         # 通话 WebSocket
├── static/
│   ├── css/
│   │   └── style.css      # 样式
│   ├── js/
│   │   ├── app.js         # 主应用逻辑
│   │   ├── chat.js        # 聊天功能
│   │   ├── call.js        # 通话功能
│   │   ├── memory.js      # 记忆管理
│   │   └── admin.js       # 后台管理
│   └── call.html          # 通话页面（独立）
├── templates/
│   └── index.html         # 主页面模板
├── models/                # 模型文件（VAD、声纹等）
├── requirements.txt       # 依赖列表
├── .env                   # 配置文件
└── README.md              # 说明文档
```

## 功能模块

### 1. 聊天功能 (routes/chat.py)
- 多会话管理（新建、切换、删除、清空）
- 多 LLM 后端支持（DeepSeek、智谱、Ollama、通义、百度、Agnes）
- Markdown 渲染
- 聊天记录搜索和导出
- TTS 语音播报

### 2. 记忆系统 (routes/memory.py)
- 保存/加载/删除记忆
- 按标签和会话过滤

### 3. 人物关系 (routes/person.py + routes/relation.py)
- 人物 CRUD
- 关系 CRUD
- AI 关系推理

### 4. 通话模式 (routes/call_ws.py + call.py)
- WebSocket 实时语音通话
- Silero VAD 语音活动检测
- ECAPA-TDNN 声纹识别
- ASR 语音转文字
- TTS 文字转语音
- **修复打断机制**

### 5. 后台管理 (routes/admin.py + static/js/admin.js)
- 服务状态监控
- LLM 配置切换
- 数据库大小显示
- 系统日志查看

## 技术选型

| 组件 | 选择 | 理由 |
|------|------|------|
| Web 框架 | aiohttp | 异步支持好，已有经验 |
| WebSocket | aiohttp + websockets | 原生支持 |
| 前端 | 原生 HTML/CSS/JS | 简单，无构建步骤 |
| 数据库 | SQLite + WAL | 轻量，已有数据 |
| TTS | edge-tts | 免费，质量好 |
| ASR | SenseVoice | 本地，免费 |
| VAD | Silero VAD | 本地，轻量 |
| 声纹 | ECAPA-TDNN | 开源方案 |
