# AI Companion

AI 语音伴侣系统 - 单服务架构版本

## 功能

- 多会话聊天
- 多 LLM 后端支持（DeepSeek、智谱、Ollama、通义、百度、Agnes）
- 记忆系统
- 人物关系管理
- TTS 语音播报
- 后台管理

## 安装

```bash
pip install -r requirements.txt
```

## 启动

```bash
python app.py
```

或双击 `启动.bat`

## 访问

浏览器打开 http://localhost:8765

## 配置

编辑 `.env` 文件配置 API Key 等参数。
